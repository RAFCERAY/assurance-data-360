## 📊 Description
Simulation complète du travail d'un Data Manager senior dans une compagnie d'assurance fictive — 71 875 lignes de données omnicanales analysées de bout en bout.

## 🎯 Missions couvertes
| Mission | Description |
|---|---|
| M1-M2 | Stratégie omnicanale & intensité relationnelle |
| M3-M4 | Analyse des frictions & personnalisation |
| M5 | Gouvernance & qualité des données |
| M6-M7-M8 | KPIs, parcours & reporting mensuel |
| M9-M10 | Machine Learning : segmentation, churn, NBA |

## 🛠️ Stack technique
Python · Pandas · Scikit-learn · Plotly · Streamlit · Jupyter

## ⚙️ Installation
```bash
pip install -r requirements.txt
python3 -m streamlit run dashboard/app.py
```
